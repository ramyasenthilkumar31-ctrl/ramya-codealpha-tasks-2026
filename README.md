# CodeAlpha UI/UX Design Internship

## Task 1 - StudyFlow Wireframe Design

## Project Overview

StudyFlow is a Smart Study Planner mobile application designed to help students manage study schedules, track assignments, monitor progress, and improve productivity through an organized learning experience.
This wireframe was created as part of the CodeAlpha UI/UX Design Internship.

## Objective

To design a user-friendly study planning application that helps students:

* Create study schedules
* Track assignments
* Monitor learning progress
* Manage academic activities efficiently

## Deliverables

* Low-Fidelity Wireframes
* User Persona
* User Flow Design
* Dashboard Layout
* Study Schedule Screen
* Assignment Tracker
* Progress Tracking Screen
* Profile Screen
